First clone the repo which has all the starting code for the COMP[29]041 JavaScript labs. 
git clone https://github.com/COMP2041UNSW/js
...
Now run the Node Package Manager (NPM) 
cd js
2041 npm install
...
If you are working locally on a CSE lab machine try starting start the Node web server: 
2041 npm start
...
You should now be able to connect to the web-server at http://127.0.0.1:8080

Working from Home

If working from home use the instructions provided by the README!.

You'll first need to install Node locally. You can find installation instructions here: https://nodejs.org/en/download/

In short, the commands are very similar, but without the 2041 CLI prefix

    cd js
    npm install
    npm start

And you're now ready to write some JavaScript.
