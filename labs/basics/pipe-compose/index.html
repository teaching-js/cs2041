Write a program pipe-compose.js which returns two functions; pipe and compose.

Pipe, should work much like the UNIX pipe. The output of the previous function should be piped into the next. Pipe should be able to take in a list of functions and execute them consecutively. You can expect valid input.

Compose, should work by 'wrapping' functions, the opposite direction to pipe. The deepest call should happen first.

To be clear; both functions need to return a function so that input can be passed in.

Your starter code should look like this.



function pipe(...args) {}

function compose(...args) {}

module.exports = {
    pipe,
    compose
}
